# O que é encapsulamento?
Agrupa dadoss e métodos que manipulam dados em uma unidade
Pode impor restriçõe ao acesso direto a variáveis e métodoss e pode vitar a modificação acidental de dados
Resumidamente -> É a proteção aos dados de um objeto

# Modificadores de acesso
Em linguagens como C++ e Java existem palavras reservadas para definir o nível de acesso aos atributos e métodos da classe
Já no python não tem palavra reservada, mas existem convenções (_) no nome do recurso 

# Definição
Público: pode ser acessado de fora da classe
Privado: Só pode ser acessado pela classe

# Properties ou propriedades 
Com o property() pode criar atributos gerenciados em suas classes, pode usar atributos gerenciados, conhecidos como propriedades e qnd precisar modificar sua implementação interna sem alterar a API pública da classe
