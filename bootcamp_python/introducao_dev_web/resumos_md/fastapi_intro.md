# Introdução FastApi
moderno framework web para construção de APIs com Python baseado em padrões como OpenAPI

traz praticidade para desenvolver

tiagem estática para validar dados e geraer documentação automática 

# Pontos positivos do FastAPI
alto desempenho
geração de documentação automática
validação de dados e serialização
facilidade de uso
suporte a async/await

# Pontos negativos do FastAPI
comunidade menor
maturidade -> novo e falta plugins
complexidade em projetos grandes
