# Banco de Dados

são coleções organizadas de dados, geralmente armazenados e acessados eletronicamente

# Tipos de BD

relacionais -> organiza os dados em tabela, tabela é composta por linhas que representam registros individuais e colunas 
não relacionais
orientado a objetos

# Papel do SGBD
Sistemas de gerenciamento de banco de dados são softwares que interagem com o usuário, outras aplicações e o proprio banco

# bancos de dados relacional

organiza os dados em tabela, tabela é composta por linhas que representam registros individuais e colunas 

tabela -> estrutura que organiza os dados em linhas e colunas, semelhante a uma planilha
cada linha = registro
cada coluna = tipo de informação

exemplo:
Tabela 'clientes' pode ter campos como 'ID', 'Nome', 'Email'

# Chave Primária 
chave primaria -> coluna ou conjunto de colunas que tem o valor único para cada registro

# Chaves estrangeiras 
chave estrangeira -> coluna ou conjunto de colunas que faz referencia a chave primaria, criando um vinculo

# Relacionamento entre tabelas
as relações podem ser

1:1
1:N (um para muitos)
N:N (muitos para muitos)
